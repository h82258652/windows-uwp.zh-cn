# <a name="media-capture-page-dev-home"></a>媒体捕获页面 （开发人员主页）
   
  
开发人员主页的**媒体捕获**页面捕获视频的游戏的当前在控制台上运行。   
 ![捕获的开发人员主页的页面](images/devhome_capture.png)   
  
**媒体捕获**页面包含以下窗格：   
 
   *  [视频捕获](#ID4EHB)  

 
<a id="ID4EHB"></a>

   

## <a name="video-capture"></a>视频捕获  
   
  
**视频捕获**窗格包含录制视频流从开发人员控制台的工具。 你还可以捕获视频的最后几秒从控制台。
