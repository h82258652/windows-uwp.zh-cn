| [字符串和二进制数据之间进行转换](convert-between-strings-and-binary-data.md) | 此示例代码显示了在 UWP 应用中，如何在字符串和二进制数据之间转换。                  |
| [向 / 从字节数组复制](copy-to-and-from-byte-arrays.md)                       | 此示例代码显示了如何在 UWP 应用中复制到字节数组并从字节数组复制。                             |
