## <a name="titleassociation"></a>TitleAssociation
 
TitleAssociation 对象具有以下规范。
 
| 成员| 类型| 说明| 
| --- | --- | --- | 
| name| 字符串| 本地化的内容的名称。| 
| id| 字符串| TitleId （32 位无符号整数、 小数中返回）。| 
| version| 字符串| （如果适用） 相关联的游戏的特定版本。| 
  
<a id="ID4E4B"></a>

 
## <a name="sample-json-syntax"></a>JSON 语法示例
 

```json
{
  "name":"Microsoft Achievements Sample",
  "id":3051199919,
  "version":"abc"
}
    
```
