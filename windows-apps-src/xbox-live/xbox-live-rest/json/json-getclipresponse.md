# <a name="getclipresponse-json"></a>GetClipResponse (JSON)
包装游戏剪辑。 
<a id="ID4EN"></a>

 
## <a name="getclipresponse"></a>GetClipResponse
 
GetClipResponse 对象具有以下规范。
 
| 成员| 类型| 描述| 
| --- | --- | --- | 
| gameClip| [GameClip](json-gameclip.md)| 满足查询单个游戏剪辑。|
