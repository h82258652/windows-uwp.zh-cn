<a id="ID4EO"></a>

 
## <a name="mediarecord"></a>MediaRecord
 
MediaRecord 对象具有以下规范。
 
| 成员| 类型| 描述| 
| --- | --- | --- | 
| id| 字符串| 使用必应目录或提供商目录的媒体的标识符。| 
| idType| 字符串| 如何解释媒体标识符。 可能的值包括"必应"和"提供程序"。| 
| name| 字符串| 媒体内容的本地化的名称。|
