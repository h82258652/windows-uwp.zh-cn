# <a name="uri"></a>/{uri}

  * [URI 参数](#ID4EX)
 
<a id="ID4EX"></a>

 
## <a name="uri-parameters"></a>URI 参数
 
| 参数| 类型| 说明| 
| --- | --- | --- | 
| <b>uri</b>| 字符串| 要使用的 URI。 请参阅方法的详细信息页面。|
